# Keymap Summary

## Vim

- Leader key: `Space`
- Insert mode: `j j` -> `Esc`
- Normal mode: `<leader>d` -> `dd`
- Normal mode: `<leader>u r` -> clear search highlight

## Navigation And Windows

- `Ctrl-h/j/k/l` -> move editor focus
- `<leader>e` -> toggle explorer
- `<leader>-` -> split below
- `<leader>|` -> split right
- `Tab` / `Shift-Tab` -> next / previous editor
- Terminal editor: `Ctrl-Tab` / `Ctrl-Shift-Tab` -> next / previous editor
- `Shift-h` / `Shift-l` -> previous / next buffer
- `<leader>w` -> save
- `<leader>t` -> create terminal editor
- `<leader>q` -> close terminal panel or current editor
- `Ctrl-w h/j/k/l` -> window focus
- `Ctrl-w s` -> split below
- `Ctrl-w v` -> split right
- `Ctrl-w c` -> close editor
- `Ctrl-w H/J/K/L` -> move active editor between groups
- `<leader>f f` -> find files
- `<leader>f g` -> grep in files
- `<leader>f t` -> create terminal editor
- `<leader><leader>` -> find files
- `<leader>/` -> grep in files
- `[ b` / `] b` -> previous / next buffer

## Code Navigation

- `<leader>c a` -> code action
- `<leader>c f` -> format
- `K` -> hover
- `<leader>c r` -> rename
- `<leader>c s` -> document symbols
- `<leader>s s` -> document symbols
- `<leader>s S` -> workspace symbols
- `g d` -> definition
- `g D` -> declaration
- `g r` -> references
- `g i` -> implementation
- `g y` -> type definition
- `[ d` / `] d` -> previous / next diagnostic

## Buffers, Git, Debug

- `<leader>b d` -> close buffer
- `<leader>b b` -> show open buffers/editors in current group
- `<leader>b o` -> close other buffers
- `<leader>s g` -> grep in files
- Search input: `Ctrl-j` / `Ctrl-k` -> move focus into the results list
- Search input: `Esc` -> return focus to active editor
- Search results list: `Ctrl-j` / `Ctrl-k` -> move down / up through results
- Search results list: `h` -> move focus back to the search input
- Search results list: `Enter` -> open selected result and return Vim to normal mode
- Search results list: `l` / `Ctrl-l` -> open selected result and return Vim to normal mode
- Search results list: `Esc` -> return focus to active editor
- `<leader>g g` -> source control
- `<leader>d a` -> start debug
- `<leader>d t` -> stop debug
- `<leader>d o` -> step over
- `<leader>d b` -> breakpoint
- `<leader>d e` -> debug hover
- `<leader>d c` -> continue

## File Explorer

- `r` -> rename
- `c` -> copy
- `p` -> paste
- `x` -> cut
- `d` -> delete
- `a` -> new file
- `Shift-a` -> new folder
- `s` -> open to side
- `Shift-s` -> open in split below
- `Enter` -> open file or expand folder

## Line Movement

- Visual mode: `Shift-j` / `Shift-k` -> move selection down / up
- Normal, Insert, Visual: `Alt-j` / `Alt-k` -> move line or selection down / up
