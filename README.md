# Portable VS Code LazyVim-Style Config

This folder contains the VS Code user settings that matter for the current
LazyVim-style workflow.

## Files

- `settings.json`: Vim-related user settings
- `keybindings.json`: custom keymaps
- `KEYMAPS.md`: human-readable keymap summary

## Usage

Copy the contents of `settings.json` into your VS Code user `settings.json`.

Copy the contents of `keybindings.json` into your VS Code user
`keybindings.json`.

## Notes

This is intentionally not a full export of your entire VS Code profile.
It only includes the parts relevant to the LazyVim-like editing experience.

The keybindings assume the `VSCodeVim` extension is installed.
